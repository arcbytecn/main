The backend for ArcByte International Web Services runs on cloud servers physically located in mainland China. Under Chinese regulations—specifically the ”Administrative Measures for the Filing of Non-commercial Internet Information Services“ and the “Administrative Measures on Internet Information Services” — any domain name pointing to a server in mainland China and providing web services via ports 80 or 443 must possess a valid ICP Filing number before going live. This requirement is based on the server's physical location, not the visitor's location. As the ICP filing for our pages intended for overseas users is currently being processed, Chinese access providers (cloud platforms/ISPs) automatically block unfiled domains at the network boundary. Consequently, the website URL may return an "Unfiled" block page or result in a connection failure. This blocking applies to all visitors (including those connecting from outside China), as traffic is ultimately routed to nodes within mainland China.

What is ICP Filing?

ICP Filing (Internet Content Provider Filing, or "Bei’an") is a non-commercial registration number issued by the Ministry of Industry and Information Technology (MIIT) of China (or provincial communications administrations) for websites, APIs, Mini Programs, and applications hosted on servers or backends located in mainland China. It is not a business license but a compliance record that links a domain name, server IP, and legal entity within the MIIT’s real-name registration system. Once approved, this information must be displayed in the website footer. This filing must be completed before web services can be offered to the public via servers in mainland China. A separate ICP Commercial License is required only if the service involves charging users online. Overseas entities cannot file directly; the application must be submitted by a Chinese-registered entity (such as a Wholly Foreign-Owned Enterprise/WFOE or a partner) or through filing channels provided by hosting platforms. After the cloud service provider completes the preliminary review, the official approval process typically takes 10–20 working days.
